Tag de Geolocalização

Tag para Titulo

Este código JavaScript utiliza a API de geolocalização do navegador para obter a posição atual do usuário e exibi-la na página web.

id="demo

A função getLocation verifica se a geolocalização é suportada pelo navegador.
Se for, solicita a posição atual e chama a função showPosition para exibir a latitude e longitude.

Se a geolocalização não for suportada, uma mensagem de erro é exibida no elemento com o id "demo".
